## Курсовой проект "Система контроля расходов автомобилиста"



## Проект будет включать в себя :

- Возможность добавления/просмотра расходов: ТО авто, штрафы, парковки, топливо и тп

- Просмотр графиков расходов по месяцам/дням (QTcreator)

- Систему авторизации для пользователеей

- User Interface

- Хранение данных при помощи баз данных

- И т.д.
